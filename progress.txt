   add SUPABASE_URL and SUPABASE_KEY in the Streamlit Cloud secrets UI.

Known limitations / v1 trade-offs
---------------------------------
- No email verification or password reset (per PRD)
- Status validation in Python only (no DB CHECK constraint)
- Add dialog uses st.dialog (requires Streamlit >= 1.35)
- Edit panel renders in the same left sidebar (Streamlit has no
  right-side sidebar API); behavior matches PRD intent
- "Click row" is implemented as an Edit button on each row, since
  Streamlit tables don't have native row-click events
